# knn_cancer_classification
Features are computed from a digitized image of a fine needle aspirate (FNA) of a breast mass. 

# Cancer Classification using k-Nearest Neighbors (kNN)

## Description
k-Nearest Neighbors (kNN) model to classify tumors as malignant or benign with feature scaling and hyperparameter tuning.

---

## Overview
This project uses the **k-Nearest Neighbors (kNN)** algorithm to classify tumors as:
- ✅ Malignant (cancerous)  
- ✅ Benign (non-cancerous)  

It demonstrates fundamental machine learning concepts like **distance-based learning**, **feature scaling**, and **model optimization**.

---
##  Problem Statement
Early cancer detection is critical for improving survival rates.

The goal is to: k-Nearest Neighbors model to classify tumors as malignant or benign with feature scaling and hyperparameter tuning.
